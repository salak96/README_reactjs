## Formatter pritter code

```
npm install --save-dev --save-exact prettier
```

- Buat folder di root

```
.prettierrc.json
```

- Masukan isi .prittierrc.json

``` {
"tabWidth": 4,
"singleQuote": true,
"jsxSingleQuote": true,
"semi": true,
"printWidth": 150,
"bracketSpacing": true,
"braceSameLine": true
}

```

- command di terminal

```
npx prettier --write . 
```

- edit package json . di script tambahkan

``` 
"format": "prettier --write .", 
```

- untuk menjalankan di terminal

```
npm run format
```

## [Install Tailwind](https://tailwindcss.com/docs/guides/create-react-app)

- Ketik kode di terminal

```
npm install -D tailwindcss postcss autoprefixer
```

- Ketik di terminal 

``` 
npx tailwindcss init -p 
```

- copy script bawah ke tailwind.config.js ke isi content
```
content: [
"./index.html",
"./src/**/*.{js,ts,jsx,tsx}",
],

```
- copy script bawah ke index.css dan hapus semua isi sebelumnya dan hapus "app.css"
```
@tailwind base;
@tailwind components;
@tailwind utilities;

```

- kemudian import './index.css'; di main.js
```
import './index.css';
```
